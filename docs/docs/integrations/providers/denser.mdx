# Denser

>[Denser](https://denser.ai) provides an enterprise-grade AI retriever designed to streamline AI integration into your applications, ensuring cutting-edge accuracy.


## Installation and Setup

You need to sign up for DenserRetriever[https://retriever.denser.ai], create a retriever, add some data and get your retriever id. 
You need the [API Key](https://retriever-docs.denser.ai/docs/api).

from langchain_community.retrievers import DenserRetriever
