# Yuque

>[Yuque](https://www.yuque.com/) is a professional cloud-based knowledge base for team 
> collaboration in documentation.


## Installation and Setup

You have to get the `Yuque` `access_token` and `api_url` on this [page](https://www.yuque.com/settings/tokens).

## Document Loader

See a [usage example](/docs/integrations/document_loaders/yuque).

```python
from langchain_community.document_loaders import YuqueLoader
```
